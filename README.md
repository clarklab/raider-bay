# Raider Bay

Godot 4.x mobile prototype for Raider Bay, rebuilt around the recovered rules and current art direction.

## How To Run

1. Install Godot 4.x from https://godotengine.org/.
2. Open this folder as a Godot project.
3. Run the main scene.

## Implemented

- Portrait-first mobile layout targeting Android.
- 7 x 8 hidden bay grid.
- Five trophy fish: Cod, Salmon, Grouper, Halibut, Tuna.
- Three depth bands: 33%, 62%, 71%, with deeper water rewarding riskier travel.
- Orthogonal and diagonal movement.
- Three-wide dock with easier entry/exit.
- Mouse/tap-first controls; keyboard shortcuts are intentionally not used.
- Separate Moves, Finds, and Casts counters.
- Fish Finder reveals whether the current square has something.
- Cast resolves the current square, with each fishing hole limited to one cast per turn.
- Live well aging and spoilage.
- Market prices randomized per game.
- Dock selling, haggling, repairs, and upgrades.
- Weather forecast and dice-based storm/hurricane damage.
- Hull sinking condition.
- Trophy turn-ins for 10 of a single fish type.
- 14-day solo test loop.
- Start screen with solo, versus-computer, and one-slot local continue.
- Simple computer captain that fishes, sells, upgrades, and raids nearby boats.
